# Digital-first GM

## Rule reference and running-the-game aid

Instead of scrolling through a PDF on your screen, you can reference all the material you need to run your games from one place while taking full advantage of the features of the digital medium.

This site uses [Just The Docs](https://github.com/just-the-docs/just-the-docs), a documentation theme layout for Jekyll. In addition, this site utilizes several plug-ins to make running and referencing the game easier:

* Dice tray which recognizes dice notation.
* Link-hovering preview windows that can be persisted by holding the SHIFT key.
* Image-map viewer for area maps that allow for zooming in and navigating to areas by clicking on the map directly.
* Header injection into the side navigation bar allowing for a built-in table of contents. 

Fonts, colors and styles for this site are based on the [Designing Dungeons](https://dungeons.hismajestytheworm.games/) course.
