# Striking
{: .no_toc }

*3rd Level Cleric Spell*

**Duration:** 1 turn

**Range:** 30’

A single weapon is enchanted:

- **Damage:** It deals an additional 1d6 damage.
- **Treated as magical:** It is able to harm monsters which are normally only harmed by magical weapons.
