# Feeblemind

*5th Level Magic-User Spell*

**Duration:** Permanent

**Range:** 240’

An arcane spell caster (e.g. a magic-user or elf within range must **save versus spells (at a -4 penalty)** or become an imbecile, unable to think clearly or cast spells.
