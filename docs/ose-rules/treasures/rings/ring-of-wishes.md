# Ring of Wishes

Grants the wearer’s wishes (see [Wishes](/docs/ose-rules/treasures/magic-items-general/#wishes)).

- **Charges:** The number of wishes granted is noted in the [Rings](/docs/ose-rules/treasures/rings/) table. Once all wishes have been used, the ring’s magic is spent.
